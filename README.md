# MovieBookingApp

A JavaScript-based movie booking application.

## Overview

This project is a movie booking application built with JavaScript, allowing users to browse movies and make bookings.

## Tech Stack

- **Language**: JavaScript (99.4%)

## Features

- Browse available movies
- Make movie bookings
- Manage bookings

## Installation

1. Clone the repository:
```bash
git clone https://github.com/pratim007/MovieBookingApp.git
cd MovieBookingApp
```

2. Install dependencies:
```bash
npm install
```

## Usage

```bash
npm start
```
